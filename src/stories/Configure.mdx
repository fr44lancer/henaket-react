# 👋 Welcome to the Henaket Design System

## !!!! Early Development Stage !!!!

UI component library based on [Henaket Design System](https://www.figma.com/community/file/1257654638425705295) and powered by **Ant Design**, **TailwindCSS**, and **Storybook
8**.

---

### 🚀 Get Started

- Use the left sidebar to explore components
- Each component has:
  - Props (Args)
  - Interactive preview
  - Usage examples

---

### 📦 Technologies

- ✅ React + TypeScript
- ✅ TailwindCSS + AntD
- ✅ Storybook 8
- ✅ Custom Theming Support

---

### 🧪 Useful Tips

- Switch to “Docs” tab for auto-generated prop tables
- Use the “Canvas” tab to interact with components
- You can copy-paste examples from each story
